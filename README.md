# 🧮 Calculator

Projeto de uma calculadora simples desenvolvida para praticar lógica de programação e manipulação do DOM.

## 📌 Sobre o projeto

Esta aplicação simula uma calculadora básica, permitindo realizar operações matemáticas essenciais de forma rápida e intuitiva.

## 🚀 Funcionalidades

- ➕ Adição
- ➖ Subtração
- ✖️ Multiplicação
- ➗ Divisão
- 🧹 Limpar visor (C)
- ⌫ Apagar último dígito (CE)

## 🛠️ Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript
